This is a Unity project using the URP rendering pipeline and the Meta Quest SDK. To run the project, install the following packages:

- Meta XR Core SDK
- Meta XR Interaction SDK
- OpenXR Plugin
- Visual Effect Graph
- GLB file import, install from git URL: https://github.com/atteneder/glTFast.git
- Occlusion shader, install from git URL: https://github.com/oculus-samples/Unity-DepthAPI.git?path=/Packages/com.meta.xr.depthapi.urp
